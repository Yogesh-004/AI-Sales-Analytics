# AI Sales Analytics Platform

An end-to-end AI-powered sales analytics system that integrates
Machine Learning, Data Engineering, and Business Intelligence.

## Features

• Sales Analytics Dashboard
• Customer Segmentation using Machine Learning
• Sales Forecasting using Time Series Models
• Automated Data Pipeline
• Interactive Visualization
• Database Integration

## Tech Stack

Python  
MySQL  
Streamlit  
Scikit-learn  
Prophet  
Plotly  
Power BI

## System Architecture

CSV Dataset
     ↓
Python Data Processing
     ↓
Machine Learning Models
     ↓
MySQL Database
     ↓
Streamlit Dashboard
     ↓
Power BI Analytics

## Machine Learning Models

Customer Segmentation
KMeans clustering is used to group customers based on:

- Sales
- Profit
- Quantity

Sales Forecasting
Facebook Prophet is used to predict future sales trends.

## Installation

Clone the repository

git clone https://github.com/YOURUSERNAME/AI-Sales-Analytics.git

Move to project folder

cd AI-Sales-Analytics

Install dependencies

pip install -r requirements.txt

Run the dashboard

streamlit run app.py
